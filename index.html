Raspador de Notas Fiscais Rondonienses

Instruções

Este aplicativo Web irá automaticamente raspar uma lista de Notas Fiscais Rondonienses e as doar para uma entidade cadastrada.

Para tal, é necessário informar o CPF, Senha e Código de Entidade, sendo que este é definido por padrão para 38 - o Lar Fraterno da Terceira Idade. A doação estará associada ao CPF bem como eventuais prêmios sorteados.

A lista de Notas Fiscais são o resultado bruto de se escanear o código QR dos cupons físicos e espaçados por linhas.
